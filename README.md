# 🍕 Concept Sito Web Pizzeria

Sito web dimostrativo e responsive realizzato per una pizzeria locale. Il progetto si concentra su un'esperienza utente (UX) fluida, caricamento rapido delle pagine e ottimizzazione per dispositivi mobili (smartphone e tablet).

## 🚀 Prova il sito live!
Puoi navigare il sito web completo e funzionante direttamente a questo link:
👉 https://pizzeriagourmetdamami.lovable.app/
---

## 🛠️ Caratteristiche del Progetto
- **Responsive Design:** Layout perfettamente adattabile a qualsiasi schermo (PC, Tablet, Smartphone).
- **Performance:** Ottimizzazione delle risorse per un caricamento istantaneo delle pagine.
- **UI/UX:** Interfaccia pulita studiata per facilitare la consultazione del menu e la prenotazione rapida.
